# track-lap-timer
STM32 FreeRTOS 赛道圈速计时器

## 项目简介
本项目为红外触发式赛道计时设备，用于智能车/小车赛道圈速统计，区分自动红外计圈、手动按键控制两种工作模式，采用消息队列隔离输入输出逻辑，运行稳定无阻塞。

## 硬件组成
- 主控：STM32F103C8T6
- 检测：E18-D80NK 红外漫反射传感器
- 显示：TM1650 四位数码管
- 输入：独立功能按键
- 外设：蜂鸣器提示、电池ADC电压采集

## 软件架构
1. 外设输入层：按键、红外、电量采样
2. FreeRTOS调度中心：输入消息队列、全局状态机、业务逻辑处理
3. 输出任务层：数码管刷新、蜂鸣器报警提示
